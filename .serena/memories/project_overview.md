# 横盘突破监控工具 - 项目概览

## 项目目的
这是一个专为macOS平台开发的股票技术分析工具，主要功能是监控A股市场的异动拉升和横盘突破形态，并在满足预设条件时发出预警通知。

## 核心功能模块
1. **AI异动拉升检测** - 实时检测涨幅增速异常的股票
2. **横盘突破分析** - 识别横盘整理后的突破机会  
3. **资金大单监控** - 监控主力资金和大单交易
4. **热门板块统计** - 统计各板块异动股票数量
5. **冠绝榜系统** - 异动股票实时排行
6. **预警系统** - 多种预警通知方式

## 技术栈
- **后端**: Python 3.8+ + FastAPI + uvicorn
- **数据处理**: pandas, numpy, aiohttp
- **前端**: 计划使用Web技术栈 (待确定具体方案)
- **数据源**: 腾讯股票API, 东方财富API, 搜狐财经API
- **数据存储**: JSON文件 + SQLite (可扩展)
