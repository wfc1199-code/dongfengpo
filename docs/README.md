### 核心算法
```python
# 热度分数计算
def calculate_hot_score(sector_data):
    change_score = min(abs(avg_change) * 10, 50)     # 涨幅分数 (最高50分)
    amount_score = min(total_amount/1e9 * 30, 30)    # 成交额分数 (最高30分)  
    anomaly_score = min(anomaly_ratio * 20, 20)      # 异动比例分数 (最高20分)
    return change_score + amount_score + anomaly_score
```
