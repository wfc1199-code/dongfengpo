# 东风破项目综合分析报告

## 项目概述

**东风破**是一个专业的A股智能分析系统，旨在通过人工智能技术和量化算法，为投资者提供实时的市场异动检测、涨停板分析、智能选股等功能支持。

## 一、项目架构分析

### 1.1 整体架构设计

项目采用**前后端分离**的架构模式：

- **前端**：React 19 + TypeScript + ECharts 5
- **后端**：Python FastAPI + 异步处理
- **通信**：RESTful API + WebSocket（计划中）
- **数据源**：Tushare + AkShare 多源融合

### 1.2 技术栈明细

#### 后端技术栈
- **Web框架**：FastAPI（异步高性能）
- **数据处理**：Pandas、NumPy
- **机器学习**：Scikit-learn、SciPy
- **异步处理**：asyncio、aiohttp
- **缓存系统**：Redis（通过redis-py）
- **数据库**：PostgreSQL（通过psycopg2）
- **任务调度**：schedule、APScheduler
- **监控**：Prometheus客户端

#### 前端技术栈
- **框架**：React 19.0.0
- **语言**：TypeScript 5.7
- **UI库**：Ant Design 5.x
- **图表**：ECharts 5 + echarts-for-react
- **状态管理**：Zustand
- **HTTP客户端**：Axios
- **构建工具**：Create React App

### 1.3 模块划分

后端模块结构：
```
backend/
├── api/          # API路由层（24个路由模块）
├── core/         # 核心业务逻辑（34个核心模块）

前端组件结构：

```

**策略体系**：

3. **突破策略**：价格突破、成交量突破、形态突破
4. **聪明钱策略**：主力资金、机构买入、北向资金
