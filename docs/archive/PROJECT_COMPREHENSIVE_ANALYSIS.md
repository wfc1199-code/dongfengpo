## 执行摘要

东风破是一个功能完整的A股智能分析系统，专注于股票异动检测、涨停板追踪和智能选股。项目采用现代化的前后端分离架构，集成了多种数据源和智能算法，为股票投资者提供实时的市场分析和交易决策支持。

**核心亮点:**
- 🎯 **实时异动检测**: 3分钟提前预警，多维度异动识别
- 📊 **涨停板分析**: "一进二"策略专用，精准晋级率预测
- 🤖 **智能选股系统**: AI多策略融合，自动筛选优质标的
- 📈 **专业图表**: ECharts驱动的K线图、分时图等金融图表
- 🔄 **版本管理**: 内置快照系统，支持配置回滚和历史追踪

- **提前预警机制**: PRE_VOLUME_SURGE、PRE_BIG_ORDER等预警类型

**核心算法**:
```python
def detect_early_warning(self, stock_data):
    # 成交量预警检测
    recent_volumes = stock_data['volumes'][-3:]  # 最近3分钟
    historical_avg = stock_data['volume_avg_20d']
    
    # 连续3分钟成交量超过均值2倍
    if all(v / historical_avg > 2.0 for v in recent_volumes):
        return AnomalySignal(
            type="PRE_VOLUME_SURGE",
            confidence=85,
            message="成交量持续放大，可能即将拉升"
        )
```

**业务价值**:
- 提前预警潜在机会，抢占先机
- 多信号验证，降低假阳性
- 历史回测验证，算法可靠性高

```python
def calculate_promotion_probability(self, stock):
    score = 50  # 基础分50分
    
    # 封板时间权重 (40%)
    if stock.seal_time <= "10:00":
        score += 20  # 早盘强封板
    elif stock.seal_time <= "13:00":
        score += 10  # 上午封板
    
    # 封单强度权重 (30%)
    if stock.seal_ratio > 10:
        score += 15  # 封单压力大
    
    # 烂板次数权重 (20%)
    score -= stock.open_count * 5  # 每次烂板扣5分
    
    # 题材热度权重 (10%)
    if stock.theme in hot_themes:
        score += 8
    
    return min(100, max(0, score))
```

**数据准确性优化**:
- 历史数据验证：使用AkShare获取真实连板天数
- 并发处理：3个并发请求，2秒超时控制

### 2.3 智能选股系统 ⭐⭐⭐⭐

**功能描述**: AI多策略融合选股，自动发现投资机会

**四大核心策略**:

1. **AI综合策略** (权重最高)
   - 技术面：RSI、MACD、布林带等30+指标
   - 资金面：主力资金、北向资金、机构买入
   - 情绪面：市场热度、题材活跃度
   - 基本面：ROE、净利润增长率

2. **动量策略**
   - 强势突破：价格突破重要阻力位
   - 量价配合：成交量与价格良性互动
   - 趋势跟踪：均线多头排列

3. **突破策略**
   - 形态突破：三角形、矩形等图形突破
   - 关键位突破：年线、半年线突破
   - 成交量突破：放量突破重要压力

4. **聪明钱策略**
   - 主力资金追踪
   - 龙虎榜数据分析
   - 机构调研热度

**综合评分模型**:
```python
def calculate_ai_score(self, stock):
    weights = {
        'technical': 0.30,    # 技术面 30%
        'money_flow': 0.30,   # 资金面 30%
        'sentiment': 0.20,    # 情绪面 20%
        'fundamental': 0.20   # 基本面 20%
    }
    
    total_score = 0
    for dimension, weight in weights.items():
        dimension_score = self.calculate_dimension_score(stock, dimension)
        total_score += dimension_score * weight
    
    return round(total_score, 1)
```

**实现细节**:
```python
class UnifiedVersionManager:
    def create_snapshot(self, description=""):
        snapshot = {
            'id': str(uuid.uuid4()),
            'timestamp': datetime.now().isoformat(),
            'config': self.get_current_config(),
            'description': description,
            'auto_created': True
        }
        self.save_snapshot(snapshot)
        return snapshot['id']
    
    def rollback_to_version(self, version_id):
        snapshot = self.load_snapshot(version_id)
        self.apply_config(snapshot['config'])
        return True
```
