# TradingView 支撑压力指标快速安装指南

## 🚀 5分钟快速安装

### 步骤1：打开TradingView
1. 访问 [TradingView.com](https://www.tradingview.com)
2. 登录您的账户
3. 选择要分析的股票或交易对

### 步骤2：打开Pine编辑器
1. 点击页面底部的"Pine编辑器"按钮
2. 或者按快捷键 `Alt + P`

### 步骤3：复制代码
1. 打开 `TradingView_支撑压力指标_简化版.pine` 文件
2. 全选并复制所有代码（Ctrl+A, Ctrl+C）

### 步骤4：粘贴代码
1. 在Pine编辑器中删除默认代码
2. 粘贴复制的代码（Ctrl+V）

### 步骤5：保存指标
1. 点击"保存"按钮
2. 输入指标名称：`支撑压力系统`
3. 点击"确定"

### 步骤6：添加到图表
1. 点击"添加到图表"按钮
2. 选择1分钟时间框架
3. 等待指标加载完成

## 📊 指标解读

### 线条颜色
- **蓝色实线**：固定阻力位
- **红色实线**：固定支撑位
- **灰色虚线**：中轴线
- **橙色虚线**：动态阻力位
- **绿色虚线**：动态支撑位

### 交易信号
- **绿色向上三角形**：买入信号
- **红色向下三角形**：卖出信号

## 🔧 常见问题解决

### 问题1：指标不显示
**解决方案**：
- 确保选择1分钟时间框架
- 等待足够的数据加载
- 检查参数设置是否正确

### 问题2：信号过多
**解决方案**：
- 增加最小触及次数到4或5
- 减小价格容忍度到0.001
- 增加计算周期

### 问题3：信号过少
**解决方案**：
- 减少最小触及次数到2
- 增加价格容忍度到0.003
- 减少计算周期
