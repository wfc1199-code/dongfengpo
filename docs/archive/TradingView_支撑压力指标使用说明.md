# TradingView 支撑压力指标使用说明

## 一、指标概述

这是一个专为TradingView 1分钟图设计的综合支撑压力指标，集成了三种不同的算法：
- **通达信固定算法**：基于价格区间的固定比例计算
- **动态支撑压力算法**：通过极值点识别和价格聚类
- **成交量支撑压力算法**：基于成交量百分位和ATR

## 二、安装步骤

### 2.1 复制代码
1. 打开TradingView网站
2. 点击"Pine编辑器"
3. 复制`TradingView_支撑压力指标.pine`文件中的完整代码
4. 粘贴到Pine编辑器中

### 2.2 保存和添加
1. 点击"保存"按钮
2. 给指标命名（如"支撑压力系统"）
3. 点击"添加到图表"
4. 选择1分钟时间框架

## 三、参数设置

### 3.1 固定支撑压力参数
```
计算周期(分钟): 240
- 用于计算固定支撑压力的周期长度
- 默认240分钟（4小时）
- 范围：60-1440分钟

- 是否显示固定支撑压力线
```

### 3.2 动态支撑压力参数
```

- 默认20个周期
- 范围：5-100

最小触及次数: 3
- 形成支撑压力位的最小触及次数
- 默认3次
- 范围：2-10

价格容忍度: 0.002
- 价格聚类时的容忍度（0.2%）
- 默认0.002
- 范围：0.001-0.01

显示动态支撑压力线: true
- 是否显示动态支撑压力线
```

### 3.3 成交量支撑压力参数

```
- 用于计算成交量支撑压力的周期
- 默认25个周期
- 范围：10-100

- 形成阻力位的成交量百分位阈值
- 默认80%
- 范围：50-95%

支撑位成交量阈值(%): 80
- 形成支撑位的成交量百分位阈值
- 默认80%
- 范围：50-95%

显示成交量支撑压力线: true
- 是否显示成交量支撑压力线
```

### 3.4 强度评级参数

## 四、指标解读

### 4.1 线条颜色说明

#### 固定支撑压力线
- **蓝色实线**：固定阻力位（7/8位置）
- **红色实线**：固定支撑位（0.5/8位置）
- **灰色虚线**：中轴线（1/2位置）

#### 动态支撑压力线
- **绿色线**：动态支撑位
- **橙色线**：动态阻力位
- **实线**：S级或A级（强支撑压力位）
- **虚线**：B级、C级或D级（弱支撑压力位）

#### 成交量支撑压力线
- **紫色线**：成交量支撑位
- **黄色线**：成交量阻力位
- **点线**：所有成交量支撑压力位

### 4.2 强度评级系统

| 等级 | 颜色 | 说明 | 交易建议 |
|------|------|------|----------|
| S级 | 红色 | 极强支撑压力位 | 重点关注，强信号 |
| A级 | 橙色 | 强支撑压力位 | 重要参考，中等信号 |
| B级 | 黄色 | 中等支撑压力位 | 一般参考，弱信号 |
| C级 | 绿色 | 弱支撑压力位 | 次要参考 |
| D级 | 灰色 | 极弱支撑压力位 | 仅供参考 |

### 4.3 交易信号

#### 买入信号（绿色向上三角形）
- 价格接近S级支撑位
- 在支撑位下方出现
- 建议在支撑位附近买入

#### 卖出信号（红色向下三角形）
- 价格接近S级阻力位
- 在阻力位上方出现
- 建议在阻力位附近卖出

## 五、使用策略

### 5.1 支撑位交易策略

#### 买入条件
1. 价格接近S级或A级支撑位
2. 出现买入信号
3. 成交量放大确认
4. 出现反转K线形态

#### 止损设置
- 支撑位下方1-2%
- 或跌破支撑位立即止损

#### 目标价位
- 下一个阻力位
- 或支撑位上方3-5%

### 5.2 阻力位交易策略

#### 卖出条件
1. 价格接近S级或A级阻力位
2. 出现卖出信号
3. 成交量萎缩
4. 出现反转K线形态

#### 止损设置
- 阻力位上方1-2%
- 或突破阻力位立即止损

#### 目标价位
- 下一个支撑位
- 或阻力位下方3-5%

### 5.3 突破策略

#### 向上突破
1. 价格突破阻力位并站稳
2. 成交量显著放大
3. 旧阻力位变为新支撑位
4. 可考虑追涨

#### 向下突破
1. 价格跌破支撑位
2. 成交量放大确认
3. 旧支撑位变为新阻力位
4. 可考虑追跌

## 六、信息面板

### 6.1 支撑压力位表格
指标右上角显示信息表格，包含：
- 支撑压力位类型和价格
- 对应的强度评级
- 最多显示5个最重要的支撑压力位

### 6.2 实时更新
- 支撑压力位实时计算和更新
- 强度评级动态调整
- 交易信号及时出现

## 七、警报设置

### 7.1 自动警报
当启用警报时，指标会自动发送以下警报：
- **买入信号**：价格接近强支撑位
- **卖出信号**：价格接近强阻力位

### 7.2 手动设置警报
1. 右键点击图表
2. 选择"添加警报"
3. 选择"支撑压力系统"
4. 设置警报条件
5. 选择通知方式

## 九、常见问题

### 9.1 指标不显示
- 检查时间框架是否为1分钟
- 确认参数设置正确
- 等待足够的数据加载

### 9.2 信号过多或过少
- 调整价格容忍度参数
- 修改最小触及次数
- 调整成交量阈值

### 9.3 支撑压力位不准确
- 增加计算周期
- 调整极值点识别窗口
- 检查市场环境是否适合
