# 期权模块归档 (Options Module Archive)

> **状态**: 已废弃 (Deprecated)  
> **归档日期**: 2025-12-16

## 说明

此目录包含已从项目核心功能中移除的期权相关代码和文档。由于期权数据源不稳定，期权模块已从生产系统中移除。

这些文件仅作为历史参考保留，**不会**在正常启动路径中自动执行或使用。

## 归档内容

| 文件                           | 说明                                                               |
| ------------------------------ | ------------------------------------------------------------------ |
| `probe_options.py`             | AkShare 期权数据探测脚本                                           |
| `probe_alternative_options.py` | 替代期权数据源探测脚本                                             |
| `list_ak_options.py`           | AkShare 期权列表脚本                                               |
| `verify_option_impl.py`        | 期权实现验证脚本（引用已删除的 `CSI1000OptionDataSource`，已失效） |

## 移除原因

1. **数据源不稳定**: AkShare 和 Tushare 的期权数据接口存在可用性问题
2. **非核心功能**: 当前主要业务聚焦于股票异动、涨停板监控等功能
3. **维护成本**: 期权模块增加了系统复杂度，但使用率较低

## 如需恢复

如果将来需要恢复期权功能，这些文档可作为设计参考。但建议：

1. 重新评估数据源可用性
2. 参考 `OPTION_DATA_SOURCES_RECOMMENDATION.md` 选择合适的数据提供商
3. 根据当前系统架构重新设计集成方案
