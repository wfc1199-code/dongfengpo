# 二板候选股票筛选条件详解

## 概述
二板候选股票是指今日涨停的股票中，具备明日继续上涨（二进三）潜力的标的。系统通过多维度指标对涨停股票进行筛选和评分，选出最优质的候选。

## 核心筛选条件

### 1. 基础门槛条件
| 指标 | 条件 | 说明 |
|------|------|------|
| **涨跌幅** | ≥ 8.5% | 必须接近涨停或已涨停 |
| **成交额** | ≥ 1000万元 | 确保有足够的市场关注度 |
| **换手率** | 0.5% - 20% | 换手率过低或过高都不利于二板 |
| **烂板次数** | ≤ 3次 | 开板次数过多说明封板不稳 |

### 2. 高级筛选条件
| 指标 | 优选条件 | 加分项 |
|------|----------|-------|
| **封板时间** | 10:00前 | 早盘封板概率更高 |
| **封单金额** | ≥ 0.3亿元 | 资金力量强大 |
| **成功概率** | ≥ 60% | 由算法综合计算 |

## 概率计算算法

### 基础概率 = 50分

### 加分项：

#### 1. 封板时间加分
- **09:00-10:00**: +25分（早盘封板）
- **10:00-11:30**: +15分（上午封板）
- **11:30-14:00**: +10分（中午封板）
- **14:00以后**: -10分（尾盘封板风险高）

#### 2. 封单金额加分
- **≥ 5亿**: +20分（超大封单）
- **3-5亿**: +15分（大封单）
- **2-3亿**: +10分（中等封单）
- **1-2亿**: +5分（小封单）

#### 3. 封单比例加分
- **≥ 20%**: +15分（封单占比高）
- **15-20%**: +10分（封单较强）
- **10-15%**: +5分（封单一般）

#### 4. 烂板次数扣分
- **每次开板**: -10分

#### 5. 量比加分
- **≥ 5倍**: +10分（巨量）
- **3-5倍**: +5分（放量）
- **2-3倍**: +3分（温和放量）

#### 6. 换手率评分
- **3-8%**: +10分（适中换手）
- **>15%**: -10分（换手过高）

### 最终概率范围：20% - 95%

## 题材识别算法

系统根据股票名称自动识别题材：

| 关键词 | 题材分类 |
|--------|----------|
| 科技、软件、信息、数据、智能 | **科技股** |
| 新能源、电池、锂、光伏 | **新能源** |
| 医药、生物、健康、医疗 | **医药生物** |
| 地产、建筑、基建 | **房地产** |
| 其他 | **热点题材** |

## 推荐理由生成

### 基础理由
- **涨幅 ≥ 9.8%**: "强势涨停"
- **涨幅 ≥ 9.5%**: "涨停封板"  
- **涨幅 < 9.5%**: "接近涨停"

### 附加理由
- **成交额 ≥ 2亿**: "成交活跃"
- **封板时间 ≤ 10:00**: "早盘封板"

### 示例
> "强势涨停；成交活跃；早盘封板"

## 风险提示生成

### 自动识别风险
- **换手率 > 10%**: "换手率偏高"
- **涨幅 < 9.5%**: "未完全涨停"

### 通用风险
- "市场波动风险"
- "政策变化风险"

### 最多显示3个风险点

## 评分体系

### 技术评分 (60-95分)
基于股票的技术指标和涨停强度

### 市场评分 (60-95分)  
基于成交量、换手率等市场表现

### 资金评分 (50-90分)
基于封单金额、资金流向等

## 排名机制

1. **按概率排序**: 概率高的排名靠前
2. **概率相同时**: 按封单金额排序
3. **最多选择**: 15只候选股票
