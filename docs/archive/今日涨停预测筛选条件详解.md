# 今日涨停预测筛选条件详解

## 概述
今日涨停预测系统通过实时分析市场数据，运用多维度算法筛选出最有可能涨停的股票。与二板候选不同，这是**预测性系统**，旨在在股票涨停前提前发现机会。

## 🎯 数据来源

### 主要数据源
- **东方财富API**: `http://push2.eastmoney.com/api/qt/clist/get`
- **备用数据源**: 新浪财经API
- **实时更新**: 缓存时间60秒，确保数据新鲜度

### 数据获取范围
- **市场范围**: 沪深A股全市场（包括主板、创业板、科创板）
- **更新频率**: 每分钟更新一次
- **数据量**: 返回前50只最有潜力的股票

## 🔍 基础筛选条件

### 1. 涨幅范围筛选
```
1.0% ≤ 涨跌幅 < 9.5%
```
- **排除已涨停股票**: 避免追高风险
- **包含低涨幅股票**: 捕捉早期机会
- **分层筛选逻辑**:
  - **1%-4%**: 需要强量能支撑（量比≥1.5倍 且 换手率≥2%）
  - **5%-9%**: 保持原有筛选逻辑

### 2. 基础过滤条件
- ❌ **ST股票**: 自动过滤风险股
- ❌ **停牌股票**: 成交量为0的股票
- ❌ **成交量过小**: 低于最小成交量要求
- ❌ **换手率过低**: 缺乏市场活跃度

## 📈 推荐理由生成逻辑

系统自动生成推荐理由，基于以下规则：

### 技术面理由
- **"技术突破"**: 价格位置≥80% + 涨幅≥5%
- **"资金流入"**: 量比≥2.0倍 + 成交额≥2亿
- **"强势拉升"**: 涨幅≥6% + 量比≥1.5倍
- **"温和上涨"**: 涨幅3-6% + 换手率适中

### 量能面理由
- **"放量突破"**: 量比≥3.0倍
- **"活跃交易"**: 换手率≥5%
- **"资金关注"**: 成交额≥5亿

### 形态面理由
- **"形态突破"**: 基于分时图形态分析
- **"蓄势上涨"**: 横盘后的突破信号

## 💡 使用建议

### 1. **时间选择**
- **最佳时机**: 9:30-10:30（开盘后1小时）
- **次佳时机**: 13:30-14:00（午后开盘）
- **避免时段**: 11:00-11:30、14:30-15:00

### 2. **风险控制**
- **仓位控制**: 单股不超过总资金5%
- **止损设置**: -3%止损，避免深套
- **止盈策略**: 8-9%止盈，避免贪婪

### 3. **选股策略**
- **优先级**: 极高>高>中等
- **分散投资**: 不要集中单一题材
- **跟踪观察**: 建立自选股池跟踪

### 4. **市场环境**
- **牛市**: 可适当激进，提高仓位
- **熊市**: 谨慎参与，降低仓位
- **震荡市**: 快进快出，见好就收

## ⚠️ 风险提示

1. **预测局限性**: 无法保证100%准确，存在预测失误风险
2. **市场风险**: 大盘急跌时个股难以独善其身
3. **黑天鹅事件**: 突发消息可能导致预测失效
4. **流动性风险**: 小盘股可能存在流动性不足问题
5. **政策风险**: 监管政策变化影响市场情绪
