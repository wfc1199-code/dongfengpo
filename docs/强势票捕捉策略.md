# 东风破系统 - 强势票捕捉策略完整指南

## 📊 **核心策略概述**

东风破系统采用**三层漏斗式筛选**策略，从市场全貌到个股精选，层层递进捕捉强势票：

```
市场全貌 → 热门板块 → 强势个股 → 精准捕捉
  5000+     20个        50只       Top10
```

---

## 🔥 **第一层：热门板块智能筛选**

### 筛选标准（多维度评判）

| 维度 | 标准 | 权重 | 说明 |
|-----|------|------|------|
| **板块涨幅** | >0.5% | 40% | 基础门槛，避免下跌板块 |
| **综合热度** | >30分 | 30% | 基于涨幅、成交额、异动比例计算 |
| **参与度** | 上涨股占比>40% | 20% | 避免个别股票拉升整个板块 |
| **成交活跃度** | >50亿元 | 10% | 确保有足够资金关注 |

### 热度分数计算公式

```python
热度分数 = 涨幅分数(最高50分) + 成交额分数(最高30分) + 异动比例分数(最高20分)

涨幅分数 = min(abs(板块涨幅) * 10, 50)
成交额分数 = min(总成交额 / 10亿 * 30, 30)  
异动比例分数 = min((异动股数 / 总股数) * 20, 20)
```

### 优选结果
- 每次筛选出**8个最强势板块**
- 动态更新周期：**10分钟**
- 实际案例：铜缆高速连接(+3.5%)、重组蛋白(+3.1%)、CPO概念(+2.56%)

## ⚡ **第二层：强势个股精准筛选**

### 股票强势评分系统

| 评分项目 | 权重 | 评分标准 |
|---------|------|----------|
| **涨幅异动** | 40% | >7%:100分, >4%:70分, >2%:40分, >0.8%:20分 |
| **换手率** | 25% | >10%:60分, >5%:40分, >2%:25分 |
| **成交额** | 20% | >10亿:50分, >5亿:30分, >1亿:15分 |
| **形态强势** | 10% | 收盘位置>90%:30分, >80%:20分 |
| **量价配合** | 5% | 涨幅>3%且换手>3%:25分 |

### 筛选策略

```python
强势股条件 = (
    综合评分 >= 60分 OR
    龙头股身份 OR  
    涨幅 > 3.0% OR
    (涨幅 > 1.0% AND 换手率 > 3.0%)
)
```

### 每个板块选择**前8只最强势股票**

## 🎯 **第三层：实时异动捕捉**

### 异动检测算法

#### 强势等级分类

| 等级 | 评分范围 | 特征 | 操作建议 |
|------|----------|------|----------|
| **强势突破** | 100+ | 涨幅>7%，大额资金，强势收高 | 关注买入 |
| **加速上涨** | 70-99 | 涨幅4-7%，高换手，量价齐升 | 谨慎观察 |
| **稳步上升** | 50-69 | 涨幅2-4%，活跃换手 | 跟踪观察 |
| **温和启动** | 30-49 | 涨幅0.8-2%，资金关注 | 观察为主 |

#### 风险控制

- **高风险**：涨幅>7%，需要注意高位风险
- **中高风险**：涨幅4-7%，关注回调
- **中等风险**：涨幅2-4%，相对安全
- **低风险**：涨幅<2%，启动阶段

## 📈 **市场情绪判断体系**

### 市场强势比例计算

```
强势比例 = 强势股数量 / 总监控股票数量 * 100%
```

### 情绪等级划分

| 情绪等级 | 强势比例 | 超强股数量 | 特征描述 |
|---------|----------|------------|----------|
| **极度活跃** | >15% | >5只 | 普涨行情，追涨谨慎 |
| **活跃向上** | 8-15% | 3-5只 | 热点活跃，精选个股 |
| **温和活跃** | 4-8% | 1-3只 | 结构性机会 |
| **平稳整理** | 1-4% | <1只 | 震荡为主 |
| **相对平静** | <1% | 0只 | 等待机会 |

## 🚀 **实战操作流程**

### 日常监控步骤

1. **盘前准备**（9:00-9:25）
   - 查看隔夜热门板块变化
   - 关注新闻热点对应板块
   - 预判当日可能的热点方向

2. **开盘观察**（9:30-10:00）
   - 观察热门板块开盘表现
   - 识别强势个股早期信号
   - 记录异动股票

3. **盘中跟踪**（10:00-14:30）
   - 实时监控强势股排行榜
   - 关注新的异动突破
   - 跟踪已关注个股走势

4. **收盘总结**（15:00-15:30）
   - 复盘当日强势股表现
   - 分析强势板块持续性
   - 制定次日关注清单

### 关键时间节点

- **9:30-9:45**：开盘15分钟，观察资金流向
- **10:00-10:30**：第一波拉升时段
- **13:30-14:00**：午后启动时段  
- **14:30-15:00**：尾盘拉升时段

## ⚠️ **风险控制要点**

### 1. 技术风险控制

- 避免追高：涨幅>8%的股票谨慎参与
- 控制仓位：单只股票不超过总仓位20%
- 及时止损：跌破买入价3%考虑止损

### 2. 系统性风险

- 市场整体下跌时，个股强势难以持续
- 政策变化对特定板块的冲击
- 突发事件导致的系统性调整

### 3. 操作纪律

- 严格按照系统信号操作
- 不因个人情绪影响决策
- 保持足够的资金储备

## 📊 **系统性能监控**

### 关键指标

- **捕捉成功率**：强势股后续表现统计
- **响应速度**：从异动到提醒的时间延迟
- **误报率**：虚假信号的比例

3. **误报控制**：减少噪音信号
4. **覆盖扩展**：增加监控股票范围
